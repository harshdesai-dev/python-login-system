# 🔐 Python Login System

A beginner-friendly Login System built using Python fundamentals.

## 📌 Project Overview

This project simulates a simple login system where users enter a username and password. The system validates the credentials, limits login attempts, and locks the account after multiple failed attempts.

## ✨ Features

- ✅ Username verification
- ✅ Password verification
- ✅ Maximum 3 login attempts
- ✅ Account lock after failed attempts
- ✅ Welcome message on successful login

## 🛠️ Technologies Used

- Python

## 📚 Concepts Used

- Variables
- User Input (`input()`)
- `if-else` Statements
- `while` Loop
- Counter Variable
- Conditional Logic

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/harshdesai-dev/python-login-system.git
```

2. Open the project folder.

3. Run the program

```bash
python login.py
```
